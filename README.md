# Custom Libraries Repository

This repository is dedicated to the libraries I develop for academic and practical projects.  
It serves as a central hub for reusable components, utilities, and data structures that can be integrated into various software engineering tasks.

## Current Content
- **Stack Implementation**  
  A basic stack data structure with core operations such as `push`, `pop`, and `peek`.  
  Designed for clarity, reusability, and easy integration into larger projects.

## Purpose
- Provide a structured collection of custom libraries.
- Support learning, experimentation, and benchmarking against global standards.
- Ensure scalability and maintainability for future projects.

## Future Plans
Additional libraries and modules will be added over time, covering topics such as:
- Queues
- Linked Lists
- Trees
- Algorithms for IoT and Machine Learning
